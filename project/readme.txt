For the code to work, it is needed to follow the same folder structure.
Set the global parameter PATH in the first cell of the document to a folder that will be a root for this project.
Run the first cell and place the zipped data in the 'data' subfolder and the requirements.txt file in the root folder of this project.

The rest of the code can be simply run cell by cell.

The trained models are stored in $PATH/models folder. The training information and predictions on test set are stored in $PATH/assets folder.
